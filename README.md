# TaskTodo

This project helps users to markdown the important task of the day with the Todo.


## Connection

The user can add extension to mozilla firefox. The app works locally.

The app can also be accessed by the link on any device - https://jayapramodh.github.io/TaskTodoFirefoxExtension/

## Home

The user invited with home page asking for user name.

## Task

The task page loads with inviting message to user and prompted to add one of top priority task. If needed, the bottom place destined with ToDo complete mini task. The task and Todos can be marked completed, and can also be removed.

The page reloads the next day without any data being stored previous day.
